# Live Air Traffic Globe

Real-time visualization of aircraft positions on an interactive 3D globe using Cesium.js and React.

## Features
- Interactive 3D globe with satellite imagery
- Real-time aircraft tracking
- Altitude-based coloring gradients
- Detailed aircraft information
- Pan, zoom, and rotate the globe

## Tech Stack
- React 18
- Cesium.js (3D globe engine)
- OpenSky Network API
- Vite
- TailwindCSS
